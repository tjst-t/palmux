Features

Everything you need,
nothing you don't

モバイルで tmux を操作するために必要な機能をすべて備え、余計なものは一切ない。

Mobile-First UI

タッチ操作に最適化されたモディファイキー、スワイプジェスチャーでスクロール＆ウィンドウ切替、ピンチでフォントサイズ変更。

Single Binary

make build 一発で完結。フロントエンドも Go バイナリに組み込まれ、デプロイはファイルを置くだけ。

Session & Window Management

ドロワー UI で全セッション＆ウィンドウを一覧。作成・削除・リネームも UI から直接操作。

File Browser & Preview

ファイルブラウジング、15 以上の言語のシンタックスハイライト、Markdown レンダリング、画像プレビューに対応。

LSP Integration

gopls, pyright, rust-analyzer など Language Server を自動検出。定義ジャンプとシンボルアウトラインで IDE ライクなナビゲーション。

PWA Support

ホーム画面に追加してネイティブアプリ感覚で使える。Service Worker によるキャッシュで高速起動。

Real-time WebSocket

PTY に直接アタッチして低遅延ストリーミング。切断時は指数バックオフで自動再接続。

Security Built-in

Bearer トークン認証、TLS サポート、パストラバーサル保護を標準装備。安心して使える。

How it works

3 steps to go

インストールして、起動して、スマホで開く。それだけ。

1

Build

リポジトリをクローンして make build を実行。シングルバイナリが生成される。

2

Run

./palmux で起動。認証トークンが表示されるので控える。

3

Connect

スマホのブラウザで URL を開く。tmux が手のひらに。

Quick Start

Ready in seconds

コピペで始められる。

terminal
# Clone the repository
$ git clone https://github.com/tjst-t/palmux.git
$ cd palmux
# Install frontend dependencies & build
$ cd frontend && npm install && cd ..
$ make build
# Run Palmux
$ ./palmux
